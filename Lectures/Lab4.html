Task2: Creating a child class

 

⍟ Instructions

Look and understand the StockItem and Food classes.

Create a new empty class called Drink that inherits from StockItem

Add attributes for Canned as a boolean and volume as an integer.

In the class add a constructor that takes four parameters:

a. Name : String

b. Price : Decimal

c. Canned : bool

d. Volume : integer

Calls the appropriate superconstruction and then instantiate this particular attributes values.

Add methods that return the Volume and Canned attributes.

Uncomment the code in the StockCalculator class that creates a set of test drinks and make sure your code constructs new drinks.
